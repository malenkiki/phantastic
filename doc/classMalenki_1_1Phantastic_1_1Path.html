static Malenki\Phantastic\Path::build (   $obj)
static

Crée un chemin selon le type d’objet passé en argument.

Selon que le type d’objet est un Tag, un File de type Post, un File de type Page ou un File de type autre, cette méthode crée le chemin, tant au niveau de la chaîne de caractères, qu’au niveau du système de fichier en créant le ou les dossiers nécessaires.

Paramètres:
mixed$objun Objet Tag ou File public

string

static Malenki\Phantastic\Path::createIndex (   $str_path)
static

Ajoute un « index.html » si le chemin se termine par un simple dossier.

Si le chemin se termine déjà par un fichier, retourne juste le chemin sans changement. Cette méthode est utile dans le cas des chemins des Posts ou des Pages, quand l’URL voulu n’a pas d’estension.
